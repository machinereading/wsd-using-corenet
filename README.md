# Korean Word Sense Disambiguation using CoreNet

## About

Korean FrameNet is a lexical database that has rich annotations to represent the meaning of text using semantic frames

## Prerequisite
* `python 3.5+`
+ `numpy`
+ `scikit-learn`
+ `pandas`
+ `nltk`
+ `pgmpy (== 0.1.2)`

## Usage
